WhisperFrame

AI-drevet videotransskription med OpenAI Whisper og intelligent frame-udtrækning. Automatisér din videoanalyse.

Hent nyeste version

Funktioner

  • AI-transskription — OpenAI Whisper tale-til-tekst med høj nøjagtighed
  • Frame-udtrækning — Udtræk nøglebilleder på de vigtige øjeblikke
  • Batch-behandling — Behandl flere videoer effektivt
  • Flere formater — Understøtter gængse videoformater via FFmpeg
  • CLI-interface — Skriptbart og nemt at automatisere

Kom hurtigt i gang

pip install -r requirements.txt
python video_transcriber.py --help
python video_frame_extractor.py --help
